

# UNIVERSE 69 — LLM-native движок для ролевых игр

**Психологическая ролевая игра через механику UNO**

**Проблема:** LLM в ролевых играх быстро теряют контекст, начинают галлюцинировать и забывают правила. Это делает игру с AI-персонажами хаотичной и неуправляемой. Разработчики тратят месяцы на промпт-инжиниринг, но модель всё равно "съезжает".

**Что делает UNIVERSE 69:** Это готовый игровой фреймворк, который **останавливает дрейф контекста и галлюцинации** через карточную механику UNO как структурный контейнер. Каждый ход привязан к строгим правилам игры — модель не может "съехать" с сюжета.

**Результат:** Вы получаете управляемую ролевую игру с AI, где контекст удерживается до 2+ часов, а персонажи остаются в роли без дополнительных промптов.

## 🧠 Научное обоснование

В классических ролевых играх люди сталкиваются с внутренней цензурой и страхом осуждения (даже со стороны AI). UNIVERSE 69 решает эту проблему с помощью **внешнего математического императива** (правил UNO).

**Формула защитного механизма:**

> *«Это не я выражаю скрытые желания — этого требует математика игры для моей технической победы»*

Когда игрок совершает провокационное действие, его психика перекладывает ответственность с «себя» на случайную карту. Это позволяет безболезненно обойти сопротивление Эго и запускает контролируемую сублимацию.

## 📊 Игровой цикл

**Выбор роли → 6 Шансов → Выкладка карты → Отыгрыш → Верификация → Добор до 6**

### Эмерджентный геймплей: пример из реальной игры

**Ситуация:** Игрок (Зажатый Бухгалтер) обязан покрыть карту и выкладывает 3♣.

**Задание:** *«Погладь себя эротично»*

**Отыгрыш:** Вместо банального штампа игрок пишет:

> *«Я испуганно смотрю на тебя, медленно расстегиваю блузку и судорожно, вслух считаю пуговицы, поглаживая ткань: 1... 2... 37... На финальном счете моя героиня доведена до ментального оргазма чистыми, сухими цифрами»*

**Вердикт:** ✅ ВЕРЮ!

Характер удержан, математическое задание выполнено. Из жестких рамок родился уникальный сюжетный перформанс.

## 🚀 Быстрый старт

### Групповая игра (2–9 человек)
1. Соберите компанию, выберите крупье
2. Раздайте каждому по **6 карт** и **6 фишек «Шанс»**
3. Каждый выбирает аватара из `play/avatars.md`
4. Следуйте правилам из `play/rules.md`
5. Говорите правду через роль: если группа не поверит — потеряете «Шанс»

### Соло-игра (1 человек)
1. Откройте `play/solo-prompt.md`
2. Скопируйте промпт полностью
3. Вставьте в ChatGPT, DeepSeek или Claude первым сообщением
4. Играйте с AI-соперником
